# ML-Diabeties_Prediction
Overview

    This project predicts whether a person has Diabetes using a machine learning classification model trained on medical data.

Files
    
    Project_3_Diabetes_Prediction.ipynb – Model training and evaluation

diabetes.csv – Dataset used for prediction

  Approach
  
    Type: Supervised Learning
    
    Problem: Binary Classification
    
    Algorithm: Logistic Regression

Libraries: NumPy, Pandas, Scikit-learn

How to Run
    
    Open the notebook in Jupyter or Google Colab
    
    Upload the dataset
    
    Run all cells

Output
    
    Predicts Diabetic or Non-Diabetic
    
    Shows model accuracy
